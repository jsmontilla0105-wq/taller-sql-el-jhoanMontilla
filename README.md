# Taller SQL 

La base de datos utilizada es `tienda.db`, la cual incluye información de clientes,
productos, ventas y detalle de ventas.

## Objetivo

Aplicar consultas SQL utilizando:
- SELECT
- JOIN
- GROUP BY
- HAVING
- CTE
- Funciones de agregación
- Manejo de fechas en SQLite

## Estructura

Cada ejercicio se encuentra en un archivo independiente:

01_ejercicio.sql  
02_ejercicio.sql  
...  
30_ejercicio.sql  

## Cómo ejecutar

1. Abrir la terminal en la carpeta del proyecto.
2. Ejecutar:

   sqlite3 tienda.db

3. Luego ejecutar el ejercicio deseado:

   .read 01_ejercicio.sql

## Notas

- Se trabajó únicamente con consultas SELECT.
- Algunas edades se calculan de manera aproximada usando el año actual.
